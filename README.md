# Unemployment Analysis using Python

## 📊 Objective
Analyze unemployment trends and understand the impact of COVID-19 using data visualization.

## 📁 Dataset
Contains unemployment data across regions and time.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Analysis Performed
- Time series analysis
- COVID-19 impact study
- Region-wise comparison
- Trend visualization

## 📌 Key Insights
- Unemployment increased during COVID-19
- Different regions show different trends
- Data helps understand economic impact
